<p>
This menu give access to an HTML representation of the bibliography.
</p>

<ul>
    <li><strong>All</strong>:  all entries are displayed by increasing id,</li>
    <li><strong>Groups</strong>: display entries of a given group,</li>
    <li><strong>Search</strong>: search the database for an entry (title, keywords, authors).</li>
</ul>

Several actions may be available for each displayed entry:

<ul>
    <li>Download a postscript <img src="data/images/stock_book-16.png" alt="ps"/>, a compressed postscript <img src="data/images/tar.png" alt="ps.gz"/>, a PDF <img src="data/images/pdf-document.png" alt="pdf"/> release of the paper.</li>
    <li><img src="data/images/stock_about-16.png" alt="abstract"/> Display the abstract.</li>
    <li><img src="data/images/stock_convert-16.png" alt="BibTeX"/> Get the BibTeX source code.</li>
    <li><img src="data/images/link-url-16.png" alt="webpage"/> Go to a related webpage (Home pages for instance).</li>
    <li><img src="data/images/stock_jump-to-16.png" alt="Local link"/> Go to a local page on the web server.</li>
    <li><img src="data/images/stock_edit-16.png" alt="edit"/> Edit the entry to update information (need authentication mode disabled or authenticated user).</li>
    <li><img src="data/images/stock_delete-16.png" alt="delete"/> Remove the entry (need authentication mode disabled or authenticated user) <span style="color:red">No Warning!!</span>.</li>
    <li><img src="data/images/cvs-add-16.png" alt="add"/> Add the entry to the basket.</li>
</ul>
